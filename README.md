pico_serial_driver.py program służący do wgrywania kodów na Pico oraz sterowania nimi. Można go włączyć z pulpitu. 

Instrukcja obsługi:
Najpierw należy wybrać port na którym jest podpięte Pico.   
Następnie wybrać plik do wgrania na płytkę.   
Wcisnąć przycisk "Wgraj na Pico" lub zaznaczyć "Auto wgraj przy Połącz", wtedy nie musimy wgrywać sami kodu, tylko za każdym razem kod wgra się samodzielnie przy połączeniu z płytką.
Połączamy się z Pico przy pomocy przycisku "Połącz"     
Kolejno naciskamy ikonke Restart lub wciskamy na klawiaturze Ctrl+D, aby zrobić soft-reset na Pico co pozwoli jej uruchomić wcześniej wgrany na nią skrypt.
Dalej możemy sterować przuciskami lub wpisywać pod wypisywanym tekstem odpowiadające tym przyciską litery i potwierdzamy Wciskając enter.

Programy:
main.py program głwony 
mainz.py lekko zmodyfikowany program głowny 

Oba programy powinny nadawać się do testowania płytki.
